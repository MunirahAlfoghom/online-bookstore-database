Online Bookstore Database System

This project is a database design for an online bookstore system.

Features
- Book inventory management
- Customer order processing
- Shipping system
- Reviews and ratings

Database Design
- ERD model
- Normalization (1NF, 2NF, 3NF)
- Relational schema

Files
- schema.sql (tables)
- data.sql (sample data)
- queries.sql (queries)

Technologies
- SQL
- Database Design
